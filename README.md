# Interactive Calculator UI

This is the user interface of a fully-featured interactive calculator, built using HTML and CSS. The layout is clean and intuitive, ready to be enhanced with JavaScript for full functionality.

## Features (UI only)
- Sleek and responsive design
- Buttons for all basic arithmetic operations
- Visually appealing layout for easy interaction

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Explore the calculator interface

## Tech Stack
- HTML
- CSS
